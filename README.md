# StarlightKernel for Guam

</div> 

What it has:

- KernelSU-Next and SUSFS (KernelSU-Next version: 32940 (legacy branch) SUSFS version: v2.0.0)

- Simple LMK

- LZ4 as default ZRAM compression algorithm

- Westwood, BBR and Vegas added (Westwood is the default)

- More to come soon (BORE scheduler, the Capacity Aware Superset Scheduler (CASS), lz4 being updated to 1.10.0, and others.)

Notes:

- This is only tested on LineageOS and AxionOS, other ROMs will work (very certainly), but i donÂ´t know if it will work on the Stock ROM.

- Compiled with Clang 18.0.1 (clang-r522817) and LLVM from AOSP clang prebuilts.

Bugs:

- None found.
